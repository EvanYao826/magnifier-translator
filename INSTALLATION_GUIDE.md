# Magnifier Translator 安装和使用说明

## 项目简介

Magnifier Translator 是一款双击选词即时翻译浏览器插件，支持离线词典和在线翻译，让外语阅读更加轻松流畅。

**核心功能**：
- 双击选词即时翻译
- 离线词典优先，在线翻译兜底
- 响应式tooltip定位
- 支持用户设置

## 支持的浏览器

本插件基于标准的Chrome扩展API开发，理论上支持所有基于Chromium的浏览器：

- ✅ Google Chrome
- ✅ Microsoft Edge
- ✅ Opera
- ✅ Brave
- ✅ Vivaldi
- ✅ 其他基于Chromium的浏览器

## 安装步骤

### 方法一：从浏览器插件市场安装（推荐）

当插件发布到各浏览器市场后，可以直接从市场搜索并安装：

1. 打开浏览器的扩展管理页面
2. 搜索 "Magnifier Translator"
3. 点击 "添加到浏览器" 按钮
4. 按照提示完成安装

### 方法二：开发者模式安装（适用于本地开发和测试）

1. **构建插件**：
   - 确保已安装 Node.js 和 npm
   - 克隆或下载项目代码到本地
   - 打开命令行工具，进入项目目录
   - 执行 `npm install` 安装依赖
   - 执行 `npm run build` 构建插件

2. **安装到浏览器**：
   - **Google Chrome**：
     1. 打开 `chrome://extensions/`
     2. 开启右上角的 "开发者模式"
     3. 点击 "加载已解压的扩展程序"
     4. 选择项目目录下的 `dist` 文件夹
     5. 插件会自动安装并显示在扩展栏中

   - **Microsoft Edge**：
     1. 打开 `edge://extensions/`
     2. 开启左侧的 "开发者模式"
     3. 点击 "加载解压缩的扩展"
     4. 选择项目目录下的 `dist` 文件夹
     5. 插件会自动安装并显示在扩展栏中

   - **Opera**：
     1. 打开 `opera://extensions/`
     2. 开启右上角的 "开发者模式"
     3. 点击 "加载已解压的扩展"
     4. 选择项目目录下的 `dist` 文件夹
     5. 插件会自动安装并显示在扩展栏中

   - **Brave**：
     1. 打开 `brave://extensions/`
     2. 开启右上角的 "开发者模式"
     3. 点击 "加载已解压的扩展程序"
     4. 选择项目目录下的 `dist` 文件夹
     5. 插件会自动安装并显示在扩展栏中

## 使用方法

### 基本使用

1. **开启插件**：确保插件已启用（点击扩展栏中的插件图标，确保 "启用" 选项已勾选）
2. **选择文本**：在网页中双击或选中任意英文单词
3. **查看翻译**：翻译结果会以tooltip形式显示在选中文本附近
4. **关闭tooltip**：点击页面其他地方或开始新的选择

### 高级设置

1. **打开设置页面**：点击扩展栏中的插件图标，选择 "选项"
2. **语言设置**：选择目标翻译语言（默认为中文）
3. **启用/禁用**：可以临时禁用插件功能

## 功能说明

### 翻译方式

- **离线翻译**：优先使用内置离线词典，无需网络连接
- **在线翻译**：当离线词典中没有结果时，自动使用Google Translate API
- **翻译缓存**：翻译结果会被缓存，提高重复查询的响应速度

### 支持的文本

- 支持英文单词和短语
- 支持单词变形（如复数、过去式等）
- 支持最大40个字符的文本

### 界面说明

- **原文**：显示选中的英文文本
- **翻译**：显示翻译结果
- **音标**：如果有音标信息，会显示在翻译结果下方
- **来源**：显示翻译来源（离线词典或在线翻译）

## 常见问题和解决方案

### 1. 插件不显示翻译结果

**可能原因**：
- 插件未启用
- 选中的文本不是英文
- 网络连接问题（在线翻译时）
- 离线词典加载失败

**解决方案**：
- 检查插件是否已启用
- 尝试选择英文单词
- 检查网络连接
- 刷新页面后重试

### 2. 翻译结果不准确

**可能原因**：
- 离线词典覆盖范围有限
- 在线翻译API返回结果可能不精准

**解决方案**：
- 尝试选择更完整的单词或短语
- 对于专业术语，可能需要使用专业词典

### 3. 插件在某些网站上不工作

**可能原因**：
- 网站使用了特殊的内容安全策略
- 网站禁止了扩展脚本的执行

**解决方案**：
- 尝试在其他网站上测试
- 检查网站的内容安全策略设置

### 4. 浏览器权限问题

**可能原因**：
- 插件需要存储权限来保存用户设置
- 在线翻译需要网络访问权限

**解决方案**：
- 确保插件已获得必要的权限
- 检查浏览器的隐私设置

### 5. npm构建命令执行失败

**可能原因**：
- 系统执行策略阻止了脚本运行
- 依赖安装不完整

**解决方案**：
- 以管理员身份运行命令行工具
- 执行 `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser` 来允许脚本运行
- 重新执行 `npm install` 确保依赖安装完整

## 性能和资源使用

- **内存使用**：插件使用轻量级设计，内存占用低
- **CPU使用**：仅在选择文本时激活，不会持续占用CPU
- **网络请求**：仅在离线词典无结果时才会发起网络请求
- **存储使用**：使用chrome.storage.local存储用户设置，占用空间极小

## 隐私和安全

- **数据收集**：插件不会收集用户的翻译历史或个人信息
- **权限使用**：仅使用必要的权限（storage和网络访问）
- **数据处理**：所有翻译请求都直接发送到Google Translate API，不会经过第三方服务器
- **离线词典**：离线词典数据存储在本地，不会上传到任何服务器

## 故障排除

如果遇到问题，可以尝试以下步骤：

1. **重新加载插件**：在扩展管理页面，点击插件的 "重新加载" 按钮
2. **清除缓存**：在插件选项页面，尝试清除翻译缓存
3. **检查控制台**：打开浏览器开发者工具，查看控制台是否有错误信息
4. **重新安装**：卸载并重新安装插件
